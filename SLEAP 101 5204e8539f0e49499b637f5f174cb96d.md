        3. `pip install sleap[pypi]==1.3.3`

    3. Create SLEAP activate, deactivate directories: 
        
        ```python
        mkdir -p $CONDA_PREFIX/etc/conda/activate.d
        echo '#!/bin/sh' >> $CONDA_PREFIX/etc/conda/activate.d/sleap_activate.sh 
        echo 'export SLEAP_OLD_LD_LIBRARY_PATH=$LD_LIBRARY_PATH' >> $CONDA_PREFIX/etc/conda/activate.d/sleap_activate.sh
        echo 'export LD_LIBRARY_PATH=$CONDA_PREFIX/lib:$LD_LIBRARY_PATH' >> $CONDA_PREFIX/etc/conda/activate.d/sleap_activate.sh
        source $CONDA_PREFIX/etc/conda/activate.d/sleap_activate.sh
        mkdir -p $CONDA_PREFIX/etc/conda/deactivate.d
        echo '#!/bin/sh' >> $CONDA_PREFIX/etc/conda/deactivate.d/sleap_deactivate.sh
        echo 'export LD_LIBRARY_PATH=$SLEAP_OLD_LD_LIBRARY_PATH' >> $CONDA_PREFIX/etc/conda/deactivate.d/sleap_deactivate.sh
        ```

- Starting a Project
    1. Select file - New Project 
    2. save as
- labelling
    - basic commands
        1. Click on *file* <top left corner> 
            1. Open *Project*
            2. Double Click on most recent file to open<check on last date & time opened>Sleap
        2. Save As a new file **before starting**
            1. Initials at the beginning
            2. Remove *Copy*
            3. Increase version # by one (i.e., v22 → v23)
        3. Determine if there are any predictions to correct in any of the most recent 4 videos 
            1. Double click on video of interest
            2. Compare Labeled Frames to Predicted Frames on bottom part of window → these numbers should match!
            3. If the labeled frames are lower than predicted, correct the instances
                1. To correct an instance, double click the computer-generated label. 
                    1. Click each node and drag it to the correct place. The node will change color from red to blue once you have clicked it.  
                    2. Click alt-right arrow to move to the next instance, and alt-left arrow to go back. This can be useful for seeing how the animal is moving. This can give context to allow you to better place the nodes. 
                    3. You can click the right and left arrows to move to each successive frame. 
                2. Note that not all nodes need to be turned green after double clicking the predicted instance - nodes can stay red if they’re in the correct place
                    - Instances will be included in the training/inference as long as they are NOT yellow and gray
        4. Run Inference on new videos until you have a batch of 3 videos with 20 predictions each
            1. To run inference on a video, **double click** the video
                1. The frame raster plot at the bottom should be empty (no black lines)
            2. Click on Predict in the Menu bar 
                1. Select Run Inference:
                    1. Training Inference Pipeline Type: multi-animal top down
                    2. Predict On: Random frames in current video (20 frames)
                    3. In each model tab (centroid & centered instance), ensure that the model selected has the most recent number of frames
                        1. You can determine the most recent model by going to the following folder and selecting the model with the highest n:
                    4. If the model under *Central Model Configuration* or *Centered Instance Model Configuration*, is different, make sure to select the most up to date model.
        5. Correct all predicted instances
